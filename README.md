## 👨‍💻 Sobre mí

Soy **desarrollador Full-Stack** con enfoque en construcción de software profesional, escalable y mantenible. Trabajo desde la arquitectura backend hasta interfaces modernas, aplicaciones móviles, bases de datos y flujos completos de desarrollo.

Me interesa crear sistemas bien diseñados, con código claro, buena estructura, rendimiento sólido y una experiencia de usuario cuidada. No solo escribo código: diseño soluciones que puedan crecer, mantenerse y evolucionar.

```txt
Backend sólido      Frontend moderno     Mobile multiplataforma
APIs escalables     Bases de datos       Arquitectura limpia
Código mantenible   Automatización       Mentalidad de producto
```
